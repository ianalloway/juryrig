# Security Policy

## Supported Versions

The table below lists the versions of juryrig that are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of juryrig seriously. If you find a security vulnerability, please do not report it in the public issue tracker.

Instead, please report vulnerabilities directly via email to **ian@allowayllc.com**.

We will acknowledge receipt of your report within 48 hours and provide a timeline for coordination and release of a fix.
